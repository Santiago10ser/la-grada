# La Grada 🏟️
**Pasión que se viste** — Tienda de camisetas de fútbol en Asunción, Paraguay.

## Stack
- FastAPI + Jinja2
- HTML / CSS vanilla
- Imágenes en Cloudinary

## Correr localmente
```bash
pip install -r requirements.txt
uvicorn main:app --reload
```
